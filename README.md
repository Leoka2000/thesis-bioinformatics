## Getting Started

### Prerequisites

This project requires the following dependencies:

- **Programming Language:** Java
- **Package Manager:** Maven
- **Java Dependencies:** Please se file pom.xml at the root folder

### Installation

Build thesis-bioinformatics from the source and intsall dependencies:

1. **Clone the repository:**

    ```sh
    ‚ùØ git clone https://github.com/Leoka2000/thesis-bioinformatics
    ```

2. **Navigate to the project directory:**

    ```sh
    ‚ùØ cd thesis-bioinformatics
    ```

3. **Install the dependencies:**
